# Interactive Tour Enquiry Dashboard

## Objective

Analyze tour enquiry data and build an interactive dashboard to visualize travel trends.

## Technologies Used

* Python
* Streamlit
* Pandas

## Features

* Display tour enquiry dataset
* Visualize most popular destinations
* Analyze peak enquiry months
* Show geographic distribution of users
* Summary statistics

## How to Run

1. Install dependencies:
   pip install streamlit pandas

2. Run:
   streamlit run dashboard.py

## Outputs

* Destination Analysis
* Monthly Enquiry Trends
* Geographic Distribution
* Interactive Dashboard
